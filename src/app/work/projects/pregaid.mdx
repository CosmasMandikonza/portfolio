# Pregait: Supporting Maternal Healthcare

## Overview
Born from personal experience and loss, Pregait addresses the critical need for better communication between expecting mothers and healthcare providers, especially in low-resource environments.

## Key Features
- **Secure Communication**
  - Direct and group chats
  - Offline notifications via SMS
  - End-to-end encryption
  - HIPAA compliance

- **Low-Resource Optimization**
  - Minimal data usage
  - Offline functionality
  - SMS-based notifications

- **User-Friendly Interface**
  - Built-in emojis and GIFs
  - Message editing and deletion
  - Intuitive navigation

## Technical Implementation
- **Frontend**: React, CSS, HTML
- **Backend**: Node.js with GetStream.io
- **Communications**: Twilio integration
- **Security**: End-to-end encryption

## Impact
- Provides vital support for expecting mothers in low-resource areas
- Ensures continuous communication with healthcare providers
- Reduces preventable complications through timely support
